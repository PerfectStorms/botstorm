# society
Realization of a modern social network with various experimental features.

## Technologies
### Backend
- Spring Boot 2.1.5
- Spring Data JPA 2.1.5
- Spring Security 2.1.5
- MySQL 8.0.15
- Lombok
- DevTools

### Frontend
- NodeJS
- ReactJS or Angular ??
- Webpack
- HTML/CSS/JS

## Run apps
#### Backend
```
$ cd backend
$ mvn spring-boot:run
```

#### Frontend
```
$ cd frontend
$ yarn start
```
