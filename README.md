# society
Realization of a modern social network with various experimental features.

## Technologies
### Backend
- Spring Boot 2.1.5
- Spring Data JPA 2.1.5
- Spring Security 2.1.5
- MySQL 8.0.15
- Lombok
- DevTools

### Frontend
#### WEB
- NodeJS
- ReactJS or Angular ??
- Webpack
- HTML/CSS/JS

#### Android
- Kotlin
- Android SDK 21

#### IOS
- Swift

#### Desktop
- Java FX

## Run apps
### Server
```
$ cd backend
$ mvn spring-boot:run
```
